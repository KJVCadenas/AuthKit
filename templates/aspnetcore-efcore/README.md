# ASP.NET Core + EF Core Auth Template

This template scaffolds a secure authentication backend using ASP.NET Core and Entity Framework Core.

## Features

- JWT access/refresh tokens (rotation, reuse detection)
- Register, verify email, login, refresh, reset password, logout
- RBAC: User/Admin/Moderator roles
- OWASP Top 10 mitigations
- Unit/integration tests
- Linting with dotnet-format

See the main AuthKit docs for usage instructions.
